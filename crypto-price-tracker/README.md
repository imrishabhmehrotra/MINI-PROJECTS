# Crypto Price Tracker


<!-- ABOUT THE PROJECT -->
## About The Project

This is designed using coingecko api. It display various cryptos along with their price, symbol, volume (24 hours), 24hr volume change % and mkt cap.

### Built With

The techs used in this are:
* HTML
* CSS
* JavaScript
* reactJS

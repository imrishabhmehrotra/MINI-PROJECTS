# Fake Address Generator

### Fake address generator is a fun web app which generates fake address in proper format each time we press the **Generate** button. It can be used in online registration of software/products where do not want to give our original correspondence address. The results are fetched from an API called [Random Data Generator](https://random-data-api.com/)

## Multifunctionality Notes App

## About The Project

A notes app with different kind of functionalities that allows to :

1. Add a note which will be stored in local storage
2. Delete note
3. Update Note
4. Star the note
5. Change theme
6. Search notes
7. Sort by importance
8. Sort by old
9. Sort by new



## Tech Stacks used

- HTML
- CSS
- JS

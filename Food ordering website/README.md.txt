
# Food Ordering website

## Overview of Food Ordering website

> _A simple website build using HTML, CSS & JavaScript for ordering your meal online from anywhere._



This application will provide you a form in which you can fill your details like name, adress,email,phone no. for ordering your meal at anytime.
